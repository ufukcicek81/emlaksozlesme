# Taşdemir Emlak Sözleşme Programı

GitHub Pages için hazır PWA paketidir.

## Yükleme
1. Bu klasördeki dosyaları GitHub reposunun ana dizinine yükleyin.
2. GitHub > Settings > Pages > Source: Deploy from a branch > Branch: main > Folder: / root > Save.
3. Link: `https://ufukcicek81.github.io/emlaksozlesme/?v=20260624-temiz-form`

## Not
Logo programın içine gömülüdür. PWA eski görünürse uygulamayı kaldırıp tekrar ana ekrana ekleyin.
